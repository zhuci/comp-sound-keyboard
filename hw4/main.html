Cindy's Lab 4 Part 1

This live coding language is modified from Prof. Santolucito's, which was a minimal command-based live coding engine. I have added 3 new features:

  1. Waveform Type: Sine, Sawtooth, Square, Triangle
  2. Rests: Pitches that are 0Hz will be as rests for the lenght specified
  3. Amplitude: Can be specified between 0 and 1, defaults to 0.75

To use the live coding engine, press the button to re-evaluate your code. This language allows you to specify a rhythmic pattern of beats by entering a space-separated list of times and pitches and amplitude (optional). You can also specify the type of waveform at the beginning of the code or it'll default to sine You can also include javascript computation in the values. As an example:

triangle 1+1@220@0.25 3@0 2@220*2 1*3@270
